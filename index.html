HTML Emails!

This is a catalog of HTML email templates. They try to take into account all the quirkiness of sending HTML emails. View source and witness the best of early millenium web design. Inline styles! Table-based layout!

The purpose of these templates are to provide a uniform style for sending emails from our applications that mimics our new bootstrap-based layouts at idfg.idaho.gov.

The following examples are available:

  • Email confirmation
  • Reset Password
  • Password Changed
  • Blank Template

Please review these and send feedback. Fork, open issues and send pull requests to https://github.com/stupiddingo/email-template.
